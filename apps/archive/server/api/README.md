Alarm Types

- 0 = Komut Gecikme Alarmları
- 1 = Ekipman Alarmı
- 2 = Operatör Uyarı komutlarında uyarı mesajı,
- 3 = Operatör Uyarı komutlarında Zaman Aşıldı
- 4 = Kilitleme
- 5 = PLC
- 6 = Yazılım
- 7 = Kalibrasyon Alarmı
- 8 = Akuple
- 9 = Manuel
- 10 = Durduruldu
- 11 = Proses Alarmları

Intervention Types

EVENTID: Code of the interventation by the operator;

- 52: command parameter value changed
- 53: comman IO changed
- 63: main command added to job order
- 64: main command deleted to job order
- 65: parallel command added to job order
- 66: parallel command deleted in job order
- 70: change in job order (such as parameter value change)
- 100: Start button was pressed on the controller
